# 🍕 Pizza Delivery & Inventory Management System

A full-stack Pizza Delivery and Inventory Management System built using the MERN Stack. This project allows users to browse pizzas, customize orders, make online payments, and track orders, while administrators can manage pizzas, inventory, users, and orders through a dedicated admin panel.

## Features

### User
- User Registration & Login
- Email OTP Verification
- Browse Pizza Menu
- Build Your Own Pizza
- Add to Cart
- Online Payment (Razorpay Test Mode)
- Order Tracking
- Order History
- Profile Management

### Admin
- Secure Admin Login
- Dashboard
- Pizza Management
- Inventory Management
- Order Management
- User Management
- Low Stock Alerts

## Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB
- Tailwind CSS
- Razorpay
- JWT Authentication
- Cloudinary

## Installation

1. Clone the repository
2. Install dependencies
3. Configure environment variables
4. Start the server
5. Start the client
